# estudoGitHub

Haverá lançamentos e atualizações aqui mas sem projeto nenhum, o único objeto desse projeto é simplesmente para treinar o Git e GitHub

Sendo assim nada será desenvolvido aqui 
git init
git status
git log
git add
git commit -m "Alguma coisa aqui"
git push -u origin master (para subir tudo lá no github)
git remote -v (pra verificar quem está logado)
git pull 

enfim, vários comando git serão colocado aqui para auxiliar iniciantes a terem conhecimento com gitHub
